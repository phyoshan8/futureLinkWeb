# FutureLink Website

FutureLink Website is a React + TypeScript + Vite application for a learning studio landing site. It includes course highlights, testimonials, upcoming events, and route-based pages for about, courses, contact, and enrollment.

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS 4
- React Router

## Installation

### Prerequisites

- Node.js 20+
- npm 10+

### Setup

1. Clone the repository.
2. Install dependencies:

```bash
npm install
```

## Run Locally

Start development server:

```bash
npm run dev
```

Open the app at `http://localhost:5173`.

## Available Scripts

- `npm run dev` - Run Vite dev server
- `npm run build` - Type-check and build for production
- `npm run preview` - Preview the production build locally
- `npm run lint` - Run ESLint

## Build

Create a production build:

```bash
npm run build
```

Preview build output:

```bash
npm run preview
```
